## Quick Start

### Option 1: Docker (Recommended)

```bash
# Pull the image
docker pull llorracc/hafiscal-qe:latest

# Run the container
docker run -it --rm llorracc/hafiscal-qe:latest

# Inside the container, reproduce the paper
./reproduce.sh --docs main
```

### Option 2: Local Installation

```bash
# Clone the repository
git clone https://github.com/llorracc/HAFiscal-QE.git
cd HAFiscal-QE

# Set up the environment
./reproduce.sh --setup

# Reproduce the paper
./reproduce.sh --docs main
```
