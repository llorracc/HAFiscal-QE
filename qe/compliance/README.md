# QE Compliance Documentation

This directory contains compliance verification documents for the Quantitative Economics submission.

## Files

- **QE-COMPLIANCE-REPORT_YYYYMMDD-HHMMh.md** - Detailed compliance verification report with evidence and recommendations
- **QE-COMPLIANCE-CHECKLIST_YYYYMMDD-HHMMh.md** - Abbreviated checklist for quick status overview
- **QE-COMPLIANCE-REPORT-LATEST.md** - Symlink to most recent detailed report
- **QE-COMPLIANCE-CHECKLIST-LATEST.md** - Symlink to most recent checklist
- **QE-COMPLIANCE-SPEC.md** - Complete QE journal requirements specification

## Usage

- Review the **LATEST** symlinks for the most recent compliance status
- Check timestamped files for historical compliance records
- Refer to **QE-COMPLIANCE-SPEC.md** for complete requirements

## Generation

These documents are generated by the automated QE submission preparation workflow.
